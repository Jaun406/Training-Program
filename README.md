# 💪 Gains Tracker

A personal mobile workout tracker built for natural athletes. No ads, no subscriptions, no accounts — just open it and train.

**Live app → [Launch Gains Tracker](https://YOUR-USERNAME.github.io/gains-tracker/)**

---

## About

Gains Tracker is a lightweight, offline-capable progressive overload tracker built as a single HTML file. It was designed around a structured 5-day training split with a specific goal of hitting a 120 kg bench press naturally.

No frameworks. No dependencies. No backend. Just HTML, CSS and vanilla JavaScript — works on any device, any browser, completely offline after first load.

---

## Features

- 📅 **5-day training split** — Push / Pull / Legs / Arms / Shoulders
- 🏋️ **Per-exercise logging** — log weight and reps for every exercise
- ✅ **Set tracking** — tap to mark each individual set as complete
- 📈 **Bench press progression** — dedicated tracking toward 120 kg goal with history log
- 📊 **Progress dashboard** — sessions logged, weekly training days, bench press timeline
- 💡 **Tips tab** — key training and nutrition reminders built in
- 📱 **Mobile first** — designed for phone use, installable as a home screen app
- 💾 **Local storage** — all data saves to your device, no account needed
- ⚡ **Offline capable** — works without internet after first load

## The Training Split

| Day | Session | Focus |
|-----|---------|-------|
| Monday | Push | Chest · Shoulders · Triceps |
| Tuesday | Pull | Back · Biceps |
| Wednesday | Legs | Quads · Hamstrings · Glutes · Calves |
| Thursday | Arms | Biceps · Triceps isolation |
| Friday | Shoulders | Shoulders · Rear delts · Core |

All sessions are designed to be completed in **45 minutes** using supersets to maximise time efficiency.

---

## Bench Press Goal

The app includes a dedicated bench press progression tracker built around a **linear → intermediate → advanced progression model**:

- Add **2.5 kg** when you hit the top of your rep range for all working sets, two sessions in a row
- Deload every 4–6 weeks — drop to 60% for one week, come back stronger
- Target timeline to 120 kg: **12–18 months** of consistent natural training

## How to Use

### On your phone (recommended)

1. Open the live link in **Safari** (iPhone) or **Chrome** (Android)
2. Tap **Share → Add to Home Screen** (iPhone) or **Menu → Add to Home Screen** (Android)
3. Name it **Gains** and add it — it will appear as an app icon
4. Open it and start logging

### In a browser

Just open the live link. Everything works in any modern browser.

## Installation (run locally)

No build steps or dependencies required:

```bash
git clone https://github.com/YOUR-USERNAME/gains-tracker.git
cd gains-tracker
open index.html
```

Or simply download `index.html` and open it directly in any browser.

## Data & Privacy

All data is stored locally on your device using `localStorage`. Nothing is sent to any server. No analytics, no tracking, no accounts. Clearing your browser data will clear your logs — export or screenshot your progress if you want to keep a record.
